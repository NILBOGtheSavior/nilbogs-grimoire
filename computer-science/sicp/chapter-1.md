# Chapter 1

## The elements of programming

The three elements of prgramming languages:

- Primitive expressions: the simplest entities the language is concerned with
- Means of combination: how compound elements are built with simpler ones
- Means of abstraction: how compound elements can be named and manipulated as units

The elements of programming languages:

- Procedures: rules for manipulating data
- Data: the 'stuff' we want to manipulate
